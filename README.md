# GenAI Projects

A collection of **Generative AI** and **Natural Language Processing (NLP)** projects developed using **Python** and **Hugging Face Transformers**. This repository demonstrates practical applications of transformer-based models for language translation and creative text generation.

## Projects Included

### English to French Translator

An AI-powered machine translation system that translates English text into French using a pre-trained Transformer model.

### Features
- English to French translation
- Transformer-based Neural Machine Translation (NMT)
- Accurate and fast translation
- Simple Jupyter Notebook implementation

### Technologies Used
- Python
- Hugging Face Transformers
- MarianMT Model
- PyTorch
- NLP

---

### AI Poetic Chatbot

A Generative AI chatbot capable of producing creative and poetic responses using transformer-based language models.

### Features
- AI-generated poetry
- Creative text generation
- Conversational chatbot
- NLP-based language understanding

### Technologies Used
- Python
- Hugging Face Transformers
- Generative AI
- NLP
- PyTorch
